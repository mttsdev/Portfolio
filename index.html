Exploração IoT: Robótica e Comunicação Bluetooth

Sobre o Projeto

Projeto prático desenvolvido na faculdade focado na integração de hardware e software para controle remoto. O objetivo foi estabelecer uma comunicação confiável entre um aplicativo mobile e um microcontrolador através do protocolo Bluetooth (RFCOMM).

Demonstração Prática

(Interface de controle via Bluetooth Serial)

Especificações Técnicas

  • Microcontrolador: Arduino Uno R3.
  • Comunicação: Módulo Bluetooth HC-05 configurado para comunicação Serial.
  • Atuadores: Motores DC controlados via Ponte H (Driver L298N).
  • Perspectiva de Segurança (IoT): Estudo prático de como os dados são transmitidos em texto claro via Bluetooth e as possíveis vulnerabilidades de interceptação em redes sem fio não criptografadas.
